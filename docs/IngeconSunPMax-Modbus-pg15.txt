3. SENDING COMMANDS (PRESET HOLDING REGISTERS) 
Using the Modbus RTU protocol preset holding registers fu nction commands 
also can be sent. On table 4 are represented command holding registers 
(between 41001 and  42000).

To send a command the command has to be written on 41001. The command 
code is represented on next table. Data has to be written on 41002 holding 
register if necessary. Writing in those two register using standard modbus (0x10 
function) desired command will be applied.  
 
After executing the command readable holding registers will be upgraded with 
new values.  
 
NOTE: Starting address of command holding registers is 41001, on terms of 
sending frames it will be 1000 on dec or 0x03E8.
