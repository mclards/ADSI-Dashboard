Dashboard User Guide
Live inverter monitoring and controls
Analytics with actual vs day-ahead trends
Alarm tracking, ACK, and event persistence
Structured exports in Excel/CSV
1. Startup and Access
  • Login is required before backend/server startup proceeds.
  • License/trial validation runs before opening the dashboard.
  • Near-expiry license warning appears as a fixed notice with upload action.
2. Inverters and Node Controls
  • Each inverter card shows AC/DC power, node table, alarm code, and last seen time.
  • Node control buttons follow actual runtime state (START/STOP/ISOLATED).
  • Bulk command panel accepts ranges like 1-13,16,23-27 and requires authorization.
3. Analytics and Forecast
  • Analytics displays total plant and per-inverter MWh trends.
  • Day-ahead forecast overlays actual data and updates with selected interval/date.
  • Weekly weather context supports forecast interpretation.
4. Alarms, Audit, and Report
  • Alarm table tracks occurred time, cleared time, duration, status, and ACK state.
  • Audit table records operator, scope, command result, and IP source.
  • Report tab summarizes daily and weekly MWh, availability, and performance metrics.
5. Export and Data Persistence
  • Exports support Excel (.xlsx default) and CSV where applicable.
  • Export files are grouped by inverter scope and data category in subfolders.
  • Export options/date selections and core settings persist across app restarts.
6. IP Config and Topology Windows
  • IP Config controls inverter IPs, polling interval, and enabled nodes.
  • Topology provides interactive status mapping, filters, and quick navigation.
  • Both windows open separately to avoid blocking main dashboard operations.
7. Keyboard and Operations
  • Ctrl+T: Open topology window.
  • Ctrl+I: Open IP configuration window.
  • Ctrl + / - / 0: Native Electron zoom in/out/reset.
