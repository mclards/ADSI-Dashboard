Forecast

Manage the active forecast source and validate external provider settings from one place, including Solcast API access or authenticated toolkit login.

Provider
Local forecasting remains the standard runtime source. Solcast can run either through the standard API or through the authenticated 5-minute toolkit chart feed.
Solcast Access
Choose `Toolkit Login` when you want to use only the Solcast chart link with your toolkit account. Choose `API Key` only when you have the formal Solcast API credentials.
Solcast API
API mode reads the standard Solcast forecast endpoint. Use this only when a formal API key and resource ID are available.
Solcast Toolkit
Enter only the plant resource ID from the Solcast toolkit URL — the full URL is constructed automatically. Toolkit mode signs in with the stored account to read the forecast feed.
